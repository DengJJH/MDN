控制继承

没有很好理解

css为控制继承提供了四个特殊的通用属性值。每个css属性都接收这些值

inherit

设置该属性会使子元素属性和父元素相同。实际上就相当于开启继承

initial

设置属性值和浏览器默认样式相同。如果浏览器默认样式中未设置且该属性使自然继承的，那么会设置为inherit

unset

将属性重置为自然值，也就是如果属性使自然继承那么就是inherit，否则和initial一样

还有一个新的属性，revert只有很少的浏览器支持

  • Default link color
  • Inherit the link color
  • Reset the link color
  • Unset the link color

理解层叠

  1. 重要程度
  2. 优先级
    1. 千位：内联样式的时候该位得一分
    2. 百位：ID选择器该位得一分
    3. 十位：选择其中的类选择器、属性选择器或者伪类则该位得一分
    4. 个位：选择其中的元素、伪元素选择器则该位得一分

    注: 通用选择器 (*)，组合符 (+, >, ~, ' ')，和否定伪类 (:not) 不会影响优先级。

  3. 资源顺序
    • 多条相同权重的规则，那么最后面的规则会被应用，可以理解为后面的规则会覆盖前面的规则。
