在网页中添加矢量图形

位图与矢量图

  • 位图使用像素网格来定义 — 一个位图文件精确得包含了每个像素 的位置和它的色彩信息。流行的位图格式包括 Bitmap (.bmp), PNG (.png), JPEG (.jpg), and GIF (.gif.)
  • 矢量图使用算法来定义 — 一个矢量图文件包含了图形和路径的定义， 电脑可以根据这些定义计算出当它们在屏幕上渲染时应该呈现的样子。 SVG 格式可以让我们创造用于 Web 的精彩的矢量图形。

位图放大会模糊，而矢量图放大不会受到影响
